# 🎹 键盘操作功能 - 快速开始

## 5分钟上手教程

### 第一步：启动服务

```bash
python app.py
```

然后在浏览器访问：http://localhost:5000

### 第二步：创建任务

1. 点击 **"添加任务"**
2. 选择 **"多时间点任务"**
3. 填写任务名称（如：测试键盘操作）
4. 点击 **"+ 添加时间点"**

### 第三步：添加操作

在"操作序列"区域，点击以下按钮添加操作：

#### 🖱️ 添加点击
```
点击 [+ 点击] 按钮
→ 输入操作描述：点击输入框
→ 输入 X 坐标：960
→ 输入 Y 坐标：540
→ 点击次数：1
```

💡 **提示：** 点击"获取坐标"按钮，10秒倒计时后自动获取鼠标位置

#### ⌨️ 添加文本输入
```
点击 [+ 输入文本] 按钮
→ 输入操作描述：输入用户名
→ 要输入的文本：admin
→ 字符间隔：0.05
```

#### 🔘 添加按键
```
点击 [+ 按键] 按钮
→ 输入操作描述：切换到密码框
→ 按键：tab
```

**常用按键：**
- `enter` - 回车
- `tab` - 切换
- `esc` - 取消
- `space` - 空格
- `ctrl+c` - 复制
- `ctrl+v` - 粘贴
- `win+d` - 显示桌面
- `alt+tab` - 切换窗口

#### ⏱️ 添加等待
```
点击 [+ 等待] 按钮
→ 输入操作描述：等待页面加载
→ 等待秒数：2
```

### 第四步：保存并测试

1. 设置执行时间（建议设置为1-2分钟后）
2. 点击 **"创建任务"**
3. 点击 **"启动调度器"**
4. 等待任务自动执行

## 📋 完整示例：打开记事本并输入

```
任务名称：打开记事本测试
任务类型：多时间点任务

时间点 1 (当前时间+2分钟)
步骤名称：打开记事本并输入
☑ 执行前回到桌面
操作间隔：0.5秒

操作序列：
1. [按键] win+r         → 打开运行对话框
2. [等待] 1秒           → 等待对话框
3. [输入] notepad       → 输入程序名
4. [按键] enter         → 执行
5. [等待] 2秒           → 等待启动
6. [输入] Hello World!  → 输入文本
```

**添加步骤：**

1. **Win+R 快捷键**
   - 操作类型：按键
   - 描述：打开运行对话框
   - 按键：`win+r`

2. **等待 1 秒**
   - 操作类型：等待
   - 描述：等待对话框打开
   - 秒数：`1`

3. **输入 notepad**
   - 操作类型：输入文本
   - 描述：输入程序名
   - 文本：`notepad`

4. **回车执行**
   - 操作类型：按键
   - 描述：执行命令
   - 按键：`enter`

5. **等待 2 秒**
   - 操作类型：等待
   - 描述：等待记事本启动
   - 秒数：`2`

6. **输入 Hello World!**
   - 操作类型：输入文本
   - 描述：输入测试文本
   - 文本：`Hello World! 这是自动输入的内容。`

## ⚠️ 常见问题

### Q: 文本输入失败？
**A:** 确保输入法处于**英文模式**，中文输入暂不支持。

### Q: 快捷键不生效？
**A:** 检查按键格式，多个键用 `+` 连接，如 `ctrl+c` 而不是 `ctrl c`。

### Q: 点击位置不准确？
**A:** 使用"获取坐标"功能精确获取位置，注意不同分辨率下坐标可能不同。

### Q: 如何调试任务？
**A:** 
1. 先设置一个近期时间（1-2分钟后）
2. 观察控制台输出
3. 调整后重新测试

# 定时点击助手 v0.7.0 发布说明

**发布日期**: 2026-06-24  
**版本类型**: 重大功能更新 🎉

---

## 🎯 核心更新

### 键盘操作支持
v0.7.0 引入**键盘操作系统**，现在可以在任务中混合使用鼠标和键盘操作，实现更复杂的自动化场景。

---

## ✨ 新增功能

### 1. 四种操作类型

#### 🖱️ 点击操作 (click)
- 单击/双击/多次点击
- 自定义点击次数和间隔
- 支持屏幕任意位置

#### ⌨️ 文本输入 (type)
- 输入任意英文文本
- 自定义字符间隔时间
- 适用于表单填写、搜索等场景
- **注意**: 仅支持英文字符

#### 🔘 按键/快捷键 (hotkey)
- **单个按键**: Enter、Tab、Esc、Space、Backspace 等
- **组合快捷键**: Ctrl+C、Ctrl+V、Ctrl+A、Alt+Tab、Win+D 等
- 涵盖所有常用键盘操作

#### ⏱️ 等待操作 (wait)
- 精确等待指定秒数
- 支持小数点精度（如 0.5 秒）
- 用于等待页面加载、动画完成等

---

### 2. 可视化操作编辑器

#### 操作卡片展示
- 每个操作独立卡片显示
- 彩色类型徽章（点击/输入/按键/等待）
- 清晰的参数显示

#### 实时参数调整
- 点击次数/间隔
- 文本内容/字符间隔
- 快捷键组合
- 等待时间

#### 操作顺序管理
- 拖拽排序（规划中）
- 插入/删除操作
- 实时预览操作序列

## 🔧 技术改进

### 后端增强

#### Clicker 类扩展 (`src/clicker.py`)
```python
# 新增方法
execute_action(action_data)      # 执行单个操作
execute_actions(actions)         # 执行操作序列
convert_clicks_to_actions(clicks) # 旧格式转换
```

#### Scheduler 适配 (`src/scheduler.py`)
- 优先检测新格式 `actions` 字段
- 自动降级到旧格式 `clicks` 字段
- 无缝向后兼容

### 前端重写

#### UI 增强 (`templates/index.html`)
- 新增操作编辑器样式
- 操作类型徽章（蓝色/绿色/橙色/紫色）
- 操作参数表单
- 空状态提示

#### 交互优化 (`static/app.js`)
- 完全重写，支持操作编辑器
- 新增操作管理函数
- 改进编辑任务加载逻辑
- 优化数据收集和提交

## 🔄 向后兼容

✅ **完全兼容旧版本任务**
- 旧版本任务（使用 `clicks` 格式）继续正常工作
- 自动降级机制确保兼容性
- **无需迁移现有任务配置**

### 数据格式

**新格式** (v0.7.0+):
```json
{
  "actions": [
    {"type": "click", "x": 500, "y": 600, "clicks": 1},
    {"type": "type", "text": "hello", "interval": 0.1},
    {"type": "hotkey", "key": "enter"},
    {"type": "wait", "seconds": 2.0}
  ]
}
```

**旧格式** (v0.6.0 及以前):
```json
{
  "clicks": [[500, 600], [700, 800]]
}
```

## 📚 使用场景示例

### 场景 1: 表单自动填写
```
1. 点击 - 用户名输入框
2. 输入文本 - "john.doe@email.com"
3. 按键 - Tab
4. 输入文本 - "password123"
5. 按键 - Enter
```

### 场景 2: 商品抢购（带刷新）
```
1. 快捷键 - Ctrl+R (刷新页面)
2. 等待 - 2 秒
3. 点击 - "立即购买"按钮
4. 等待 - 0.5 秒
5. 点击 - "确认订单"按钮
```

### 场景 3: 游戏签到
```
1. 快捷键 - Win+D (回到桌面)
2. 点击 - 游戏图标
3. 等待 - 5 秒（等待加载）
4. 点击 - "签到"按钮
5. 快捷键 - Alt+F4 (关闭窗口)
```

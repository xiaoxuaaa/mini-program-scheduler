# 快速参考卡片

## 📋 定时点击助手 v0.4.0

### 🚀 快速开始

```bash
python main.py
```

### 📖 主菜单

```
1. 添加任务           ← 配置定时点击任务
2. 查看所有任务       ← 查看任务列表
3. 删除任务           ← 删除不需要的任务
4. 启用/禁用任务      ← 切换任务状态
5. 获取鼠标坐标       ← 单独获取坐标工具
6. 启动调度器         ← 后台运行，开始监控
7. 停止调度器         ← 停止后台运行
8. 查看调度器状态     ← 查看运行状态
0. 退出              ← 优雅退出
```

### ⚡ 添加任务三种方式

#### 方式 1: 自动获取（推荐）
```
坐标 1: 2  ← 输入2，3秒后自动捕获
```

#### 方式 2: 手动输入（引导）
```
坐标 1: 1  ← 输入1
  请输入坐标: 720,850
```

#### 方式 3: 直接输入（快速）
```
坐标 1: 720,850  ← 直接输入
```

### 🎯 多次点击支持

✅ **一个任务可以包含多个点击**

```
坐标 1: 640,400   ← 点击商品
坐标 2: 640,800   ← 点击购买
坐标 3: 640,900   ← 点击确认
坐标 4: done
```

**执行效果**：
- 点击1 → 等0.5秒 → 点击2 → 等0.5秒 → 点击3

### 🔧 常用操作流程

#### 新手流程
```
1. 菜单1 → 添加任务
2. 菜单2 → 查看确认
3. 菜单6 → 启动调度器
4. 最小化窗口等待
```

#### 日常使用
```
1. 菜单8 → 查看状态
2. 菜单2 → 查看任务
3. 需要时菜单7 → 停止
4. 修改后菜单6 → 重启
```

#### 临时修改
```
1. 菜单7 → 停止调度器
2. 菜单1/3/4 → 修改任务
3. 菜单6 → 重新启动
```

### ⚠️ 重要提示

| 项目 | 说明 |
|------|------|
| 窗口位置 | 保持目标窗口在前台或可点击位置 |
| 坐标固定 | 窗口大小位置改变需重新获取坐标 |
| 后台运行 | 调度器在后台运行，不阻塞界面 |
| 点击间隔 | 默认0.5秒，足够大多数场景 |
| 任务加载 | 新增任务需重启调度器生效 |

### 🆘 常见问题

**Q: 点击位置不准？**  
A: 重新获取坐标，确保窗口大小位置不变

**Q: 任务没执行？**  
A: 检查任务已启用、时间格式正确、调度器在运行

**Q: 启动后卡住？**  
A: v0.4.0已修复，调度器在后台运行

**Q: 如何停止？**  
A: 菜单7停止调度器，菜单0退出程序

**Q: 支持多次点击吗？**  
A: 完全支持！添加任务时可以配置多个坐标

### 💡 最佳实践

1. **首次使用先测试**
   - 设置2分钟后的测试任务
   - 验证坐标准确性

2. **固定窗口**
   - 固定目标应用窗口大小和位置
   - 避免频繁调整

3. **提前准备**
   - 提前10-15分钟启动调度器
   - 确保目标应用已就绪

4. **记录说明**
   - 在任务描述中记录点击含义
   - 方便后续查看和修改

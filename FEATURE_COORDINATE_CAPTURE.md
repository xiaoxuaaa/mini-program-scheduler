# 添加任务功能改进说明

## 新功能：添加任务时集成坐标获取

### 使用方法

当你在主菜单选择 `1. 添加任务` 后，在输入坐标环节，现在有三种方式：

#### 方式 1: 自动获取坐标 ✨ 新增

```
坐标 1 (输入1/2或坐标或done): 2

>> 移动鼠标到目标位置，3秒倒计时后自动获取...
   3...
   2...
   1...
✓ 已自动获取坐标 (720, 850)
```

**使用场景**：最方便！直接在添加任务流程中获取坐标

#### 方式 2: 手动输入坐标（引导）

```
坐标 1 (输入1/2或坐标或done): 1
  请输入坐标 (x,y): 720,850
✓ 已添加坐标 (720, 850)
```

**使用场景**：已经知道坐标值，需要输入时的明确引导

#### 方式 3: 直接输入坐标（快速）

```
坐标 1 (输入1/2或坐标或done): 720,850
✓ 已添加坐标 (720, 850)
```

**使用场景**：熟练用户快速输入

## 优势

### 之前的流程
1. 打开主菜单
2. 选择 `5. 获取鼠标坐标`
3. 移动鼠标，记录坐标
4. 返回主菜单
5. 选择 `1. 添加任务`
6. 手动输入记录的坐标

❌ **痛点**：需要记忆坐标，来回切换菜单

### 现在的流程
1. 选择 `1. 添加任务`
2. 输入任务信息
3. 输入 `2` 自动获取坐标
4. 移动鼠标到目标位置
5. 自动捕获坐标

✅ **优势**：一气呵成，无需记忆坐标

## 最佳实践

### 推荐操作顺序

1. **启动程序**
   ```bash
   python main.py
   ```

2. **打开微信小程序**
   - 在另一个屏幕或窗口打开
   - 导航到目标页面

3. **添加任务（选择菜单 1）**
   - 输入任务名称和时间
   - 当提示输入坐标时：
     - 输入 `2`（自动获取）
     - 快速切换到微信，鼠标移到目标按钮
     - 等待 3 秒自动捕获
     - 重复以上步骤添加更多坐标
     - 输入 `done` 完成

4. **查看确认（选择菜单 2）**

5. **启动调度器（选择菜单 6）**

### 多坐标任务示例

**场景**：商品详情页 → 立即购买 → 确认支付

```
坐标 1: 输入 2，移动到"商品详情"按钮
坐标 2: 输入 2，移动到"立即购买"按钮  
坐标 3: 输入 2，移动到"确认支付"按钮
坐标 4: 输入 done
```

## 技巧提示

1. **提前准备好页面**
   - 在添加任务前先把微信小程序页面打开
   - 确保目标按钮可见

2. **3秒倒计时充分利用**
   - 倒计时开始后立即切换到微信窗口
   - 移动鼠标到目标位置
   - 保持不动等待捕获

3. **验证坐标准确性**
   - 第一次使用建议设置一个1-2分钟后的测试任务
   - 观察点击位置是否准确
   - 如果不准确，重新创建任务

4. **混合使用输入方式**
   - 第一个坐标用自动获取（输入2）
   - 如果后续坐标相同，可以直接输入（复制粘贴）
   - 灵活选择最方便的方式

# 定时点击助手 v0.5.0 - 升级说明

## 🎉 重大更新：Web 界面版本

项目已成功升级为 **Web 界面版本**，同时保留命令行版本。

---

## 新增内容

### 1. Web 服务（`app.py`）
- Flask 后端，提供 RESTful API
- 支持局域网访问（手机可用）
- 自动状态同步（每 3 秒更新）
- 监听地址：`0.0.0.0:5000`

### 2. 响应式 Web 界面（`templates/index.html`）
- 简洁美观的现代化设计
- 完全响应式，完美支持手机访问
- 实时状态显示（绿点动画）
- Toast 通知提示
- 模态框交互

### 3. 新增依赖
```
flask==3.0.0
flask-cors==4.0.0
```

### 4. 使用文档（`WEB_GUIDE.md`）
- 完整的 Web 版使用指南
- 手机访问步骤说明
- 常见问题解答

## 注意事项

1. **配置共享**：两个版本共享 `config/tasks.json`，可以混用
2. **调度器唯一**：同一时间只能运行一个调度器实例
3. **局域网访问**：手机需连接同一 WiFi
4. **防火墙设置**：确保允许 5000 端口访问

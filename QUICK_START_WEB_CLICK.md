# 🚀 网页抢课 - 快速开始

## 3 步完成设置

### 1️⃣ 选择任务类型
```
添加任务 → 任务类型选择 "🌐 网页抢课任务"
```

### 2️⃣ 填写基本信息
```
网页地址：http://www.talktalktalk.cn/app/page/toLecturer
执行时间：20:57
☑ 提前 5 分钟打开浏览器
```

### 3️⃣ 可视化选择按钮
```
点击 "🔍 可视化选择按钮"
→ 浏览器自动打开
→ 用鼠标点击要抢的按钮
→ 自动完成！
```

---

## ⚡ 为什么更快？

| 方式 | 速度 |
|------|------|
| 坐标点击 | 需要手动获取坐标，3-5秒 |
| 网页抢课 | 提前打开，0.5秒点击 ⚡ |

**提前打开 = 瞬间点击！**

---

## 💡 适用场景

✅ **适合用网页抢课：**
- 能在浏览器打开的网页（有 http://xxx.com 地址）
- 例如：公众号H5页面、学校选课系统、抢票网站

❌ **不适合（用坐标点击）：**
- 微信小程序
- 桌面应用软件

## 🔥 立即使用

1. 安装依赖（首次使用）：
```bash
pip install selenium webdriver-manager
```

2. 启动程序：
```bash
python app.py
```

3. 访问：http://localhost:5000

4. 添加任务 → 选择 "🌐 网页抢课任务"

5. 开始抢课！

**更新日期**: 2026-06-25

## 🎯 功能说明

新增 **"网页抢课任务"** 类型，使用 Selenium 自动化技术，通过可视化选择器直接定位网页按钮，无需手动获取坐标。

### 与坐标点击的区别

| 对比项 | 坐标点击任务 | 网页抢课任务 ⭐ |
|--------|------------|----------------|
| 适用场景 | 桌面应用、微信小程序 | 网页（H5页面） |
| 定位方式 | 手动获取坐标 | 可视化选择元素 |
| 页面滚动 | 需要手动滚动 | 自动处理 |
| 窗口位置 | 必须固定 | 任意位置 |
| 稳定性 | 中 | 高 ⭐ |
| 速度 | 快 | 更快（提前打开）⭐ |

---

## 🚀 使用步骤

### 步骤 1：创建任务

1. 点击 **"添加任务"** 按钮
2. 填写任务名称（如：外教口语抢课）
3. 选择任务类型：**"🌐 网页抢课任务（智能定位）"**

### 步骤 2：配置基本信息

```
任务名称：外教口语抢课
任务描述：周四晚上8点的口语课
任务类型：🌐 网页抢课任务

网页地址：http://www.talktalktalk.cn/app/page/toLecturer
执行时间：20:57

☑ 提前 5 分钟打开浏览器（推荐）
```

### 步骤 3：可视化选择按钮

1. 点击 **"🔍 可视化选择按钮"**
2. 系统自动打开浏览器，显示目标网页
3. 页面顶部显示提示："👆 请点击要抢的按钮"
4. 用鼠标点击你要抢的 **"未约"** 按钮
5. 按钮被红色边框高亮显示
6. 系统自动生成选择器并显示

### 步骤 4：确认并创建

1. 查看选中的元素信息
2. 点击 **"测试选择器"** 确认是否正确
3. 点击 **"创建任务"** 完成

## 📋 完整示例

### 场景：外教口语抢课

```
任务配置：
- 任务名称：外教口语抢课
- 网页地址：http://www.talktalktalk.cn/app/page/toLecturer

20:52:00  自动打开浏览器和网页

20:57:00  抢课成功！
```

```

20:57:00  瞬间点击！⚡
```

**建议：一定要勾选"提前打开浏览器"！**

## 🔧 技术原理

### Selenium 自动化

1. **打开浏览器**：使用 ChromeDriver 自动打开 Chrome
2. **注入脚本**：在页面注入 JavaScript 选择器
3. **监听点击**：捕获用户点击的元素
4. **生成选择器**：自动生成 XPath 和 CSS Selector
5. **执行点击**：到点时精确点击元素

### 优点

- ✅ 不需要坐标
- ✅ 不怕页面滚动
- ✅ 不怕窗口移动
- ✅ 自动等待元素加载
- ✅ 比坐标点击更稳定

## ⚠️ 注意事项

### 1. 浏览器要求
- 需要安装 Chrome 浏览器
- 系统会自动下载 ChromeDriver（首次使用）

### 2. 网络要求
- 必须能访问目标网页
- 首次使用需要下载 ChromeDriver（需要网络）

### 3. 任务类型选择
- **网页抢课**：仅适用于 H5 网页
- **坐标点击**：适用于桌面应用、微信小程序

### 4. 浏览器状态
- 任务执行期间，不要关闭自动打开的浏览器
- 可以最小化浏览器窗口

## 🐛 常见问题

### Q: 如何判断是否适合用网页抢课？

**A:** 如果是在浏览器中打开的页面（能看到网址），就适合用网页抢课。

判断方法：
- ✅ 有完整 URL（http://xxx.com）→ 适合
- ❌ 微信小程序 → 不适合，用坐标点击
- ❌ 桌面软件 → 不适合，用坐标点击

### Q: 点击选择器后没反应？

**A:** 等待 1-2 秒，系统会自动检测。如果还是没反应：
1. 重新点击按钮
2. 确保点击的是可点击元素（按钮、链接等）
3. 关闭浏览器重新选择

### Q: 提前打开浏览器会不会被检测？

**A:** 不会。提前打开只是加载页面，不会触发任何操作。到点才会点击。

### Q: 可以同时运行多个网页抢课任务吗？

**A:** 可以！每个任务会在独立的浏览器窗口中执行。

### Q: 选择器会失效吗？

**A:** 只要网页结构不变，选择器就不会失效。如果网页改版，需要重新选择。

## 💡 使用建议

### 1. 提前测试
在正式抢课前，提前测试一次：
- 创建任务
- 选择按钮
- 设置1-2分钟后的时间
- 测试是否能成功点击

### 2. 提前打开
一定要勾选"提前打开浏览器"，这样速度最快！

### 3. 时间设置
建议设置比实际开放时间早 3-5 秒：
- 实际开放：21:00:00
- 建议设置：20:59:57 或 20:59:58

### 4. 多任务备份
可以创建多个相同任务，时间间隔几秒，增加成功率。

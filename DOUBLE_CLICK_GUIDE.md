# 双击功能 - 使用指南

## 🎉 新增功能：支持双击

现在每个坐标都可以配置是否双击！

---

## 功能说明

### 什么是双击

在某些场景下，单击可能不够，需要双击才能触发操作：
- 打开文件/文件夹（需要双击）
- 某些应用的按钮（需要双击才响应）
- 打开桌面图标（通常需要双击）

### 如何使用

在添加坐标时，每个坐标旁边都有一个 **"☑ 双击"** 选项：
- **不勾选**：单击（默认）
- **勾选**：双击

---

## Web 界面操作

### 简单任务

1. 点击"添加任务"
2. 选择"简单任务"
3. 添加坐标
4. 在需要双击的坐标旁边 **勾选"双击"**
5. 创建任务

### 多时间点任务

1. 点击"添加任务"
2. 选择"多时间点任务"
3. 添加时间点
4. 在每个时间点中添加坐标
5. 在需要双击的坐标旁边 **勾选"双击"**
6. 创建任务

## 使用示例

### 示例 1：打开桌面应用

**场景**：从桌面双击打开微信

```
坐标：(100, 600)
☑ 勾选"双击"
```

**配置数据**：
```json
{
  "clicks": [
    [100, 600, 2]  // 第三个参数 2 表示双击
  ]
}
```

### 示例 2：混合单击和双击

**场景**：双击打开微信，然后单击小程序入口

```
坐标 1：(100, 600) ☑ 双击
坐标 2：(500, 300) ☐ 单击
```

**配置数据**：
```json
{
  "clicks": [
    [100, 600, 2],   // 双击微信
    [500, 300]       // 单击小程序入口（默认单击）
  ]
}
```

### 示例 3：小程序抢课（完整配置）

**时间点 1（20:57）进入小程序**：
- (120, 680) ☑ 双击 - 桌面微信图标
- (450, 320) ☐ 单击 - 小程序入口
- (580, 410) ☐ 单击 - 约课小程序
- (720, 530) ☐ 单击 - 约课页面

**时间点 2（21:00）刷新抢课**：
- (830, 310) ☐ 单击 - 刷新按钮
- (840, 420) ☐ 单击 - 目标课程

## 界面说明

### 坐标输入框布局

```
[  X坐标  ] [  Y坐标  ] [ ☑ 双击 ] [ × ]
```

- **X坐标**：横坐标
- **Y坐标**：纵坐标
- **☑ 双击**：勾选表示双击，不勾选表示单击
- **×**：删除这个坐标

---

## 数据格式

### 单击（默认）

```json
[100, 200]
```

### 双击

```json
[100, 200, 2]
```

第三个参数 `2` 表示点击 2 次（双击）

## 技术实现

### 前端（JavaScript）

```javascript
// 读取双击选项
const doubleClickCheckbox = item.querySelector('.double-click-checkbox');
if (doubleClickCheckbox && doubleClickCheckbox.checked) {
    clickData.push(2);  // 添加点击次数参数
}
```

### 后端（Python）

```python
def click_sequence(self, coordinates: List) -> bool:
    for coord in coordinates:
        if len(coord) >= 3:
            x, y, clicks = coord[0], coord[1], coord[2]
        else:
            x, y, clicks = coord[0], coord[1], 1
        
        self.click_at(x, y, clicks=clicks)
```

## 常见问题

### Q: 什么时候需要双击？

A: 
- 打开桌面图标（如微信、QQ）
- 打开文件夹
- 某些老旧应用的按钮
- 需要"确认"的操作

### Q: 双击和两次单击有区别吗？

A: 
有区别！
- **双击**：两次点击间隔很短（约 0.1 秒），系统识别为双击事件
- **两次单击**：添加两个相同坐标，间隔较长，系统识别为两次独立点击

### Q: 如何判断是否需要双击？

A: 
1. 手动测试一下目标操作
2. 如果需要双击才能触发，就勾选双击
3. 如果单击就能触发，不勾选

### Q: 可以三击、四击吗？

A: 
当前版本只支持单击和双击。如需多次点击，可以添加多个相同坐标。

### Q: 双击会不会太快导致失败？

A: 
不会。`pyautogui.click(x, y, clicks=2)` 会自动处理双击的时间间隔，符合系统识别双击的标准。

## 使用建议

### 桌面图标

建议使用双击：
```
☑ 双击桌面图标
```

### 应用内按钮

通常使用单击：
```
☐ 单击按钮
```

### 不确定时

先测试单击，如果不响应再改成双击。

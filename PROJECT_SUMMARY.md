## 📝 使用流程

### 快速开始

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动程序**
   ```bash
   python main.py
   ```

3. **获取坐标**
   - 选择菜单 5
   - 移动鼠标到目标位置
   - 记录坐标

4. **添加任务**
   - 选择菜单 1
   - 输入任务信息和坐标

5. **启动调度器**
   - 选择菜单 6
   - 保持程序运行

## ⚠️ 已知问题

### 1. Windows 控制台中文显示乱码
**原因**: Windows PowerShell 默认使用 GBK 编码  
**影响**: 不影响功能，仅界面显示  
**临时方案**: 
- 使用英文任务名称
- 或在 PowerShell 中运行：`chcp 65001` 切换到 UTF-8

### 2. 微信窗口检测
**说明**: 测试时未检测到微信窗口是正常的（微信未运行）  
**使用时**: 确保微信已打开即可

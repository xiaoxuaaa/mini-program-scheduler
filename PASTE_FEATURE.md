# 📋 粘贴功能使用指南

## ✨ 新增功能：粘贴操作（支持中文）

v0.7.1 新增**粘贴操作**，解决中文输入问题！

---

## 🎯 功能特点

✅ **支持中文** - 可以粘贴中文、日文、韩文等任何 Unicode 字符  
✅ **自动复制** - 自动将文本复制到剪贴板并粘贴  
✅ **安全可靠** - 执行后会恢复原剪贴板内容  
✅ **简单易用** - 一个操作完成复制+粘贴  

---

## 📖 使用方法

### 在 Web UI 中使用

1. 创建或编辑"多时间点任务"
2. 添加时间点
3. 点击 **"+ 粘贴"** 按钮
4. 填写：
   - **操作描述**：例如"粘贴密码"
   - **要粘贴的文本**：例如"你的密码123"（支持中文！）

---

## 💡 使用场景

### 场景 1：登录系统（中文密码）

```
1. [点击] (800, 300)     → 点击用户名框
2. [粘贴] admin           → 粘贴用户名
3. [按键] tab             → 切换到密码框
4. [粘贴] 你的密码123     → 粘贴密码（支持中文！）
5. [按键] enter           → 登录
```

### 场景 2：填写中文表单

```
1. [点击] (500, 400)      → 姓名框
2. [粘贴] 张三            → 输入姓名
3. [按键] tab             → 下一个字段
4. [粘贴] 北京市朝阳区    → 输入地址
5. [按键] tab             → 下一个字段
6. [粘贴] 备注信息        → 输入备注
```

### 场景 3：混合使用

```
1. [点击] (800, 300)      → 点击输入框
2. [输入] email@          → 输入英文前缀
3. [粘贴] 公司名.com      → 粘贴中文域名
4. [按键] enter           → 提交
```

## 🆚 输入 vs 粘贴

| 功能 | 输入 (type) | 粘贴 (paste) |
|------|-------------|--------------|
| **支持中文** | ❌ 仅英文 | ✅ 支持中文 |
| **速度** | 可调节字符间隔 | 瞬间粘贴 |
| **适用场景** | 英文账号、邮箱 | 密码、中文内容 |
| **模拟效果** | 逐字输入 | 直接粘贴 |

**建议：**
- **英文内容** → 用"输入"（更真实）
- **中文内容** → 用"粘贴"（唯一选择）
- **密码** → 用"粘贴"（支持特殊字符）

## ⚙️ 技术实现

粘贴功能使用 `pyperclip` 库实现：

1. 保存当前剪贴板内容（备份）
2. 将文本复制到剪贴板
3. 模拟 `Ctrl+V` 按键粘贴
4. 恢复原剪贴板内容（可选）

**依赖：**
- `pyperclip` - Python 剪贴板操作库（已安装）

---

## ⚠️ 注意事项

### 剪贴板影响
执行粘贴操作会临时使用系统剪贴板，但会在完成后恢复原内容。

### 焦点问题
粘贴时需要确保目标输入框已获得焦点，建议：
- 先用"点击"操作点击输入框
- 再用"粘贴"操作

### 特殊字符
支持所有 Unicode 字符，包括：
- ✅ 中文、日文、韩文
- ✅ emoji 表情 😊
- ✅ 特殊符号 @#$%^&*

---

## 🐛 故障排除

### Q: 粘贴后内容不对？
**A:** 可能是焦点不在目标输入框，添加"点击"操作先聚焦。

### Q: 粘贴失败？
**A:** 检查是否有其他程序占用剪贴板，关闭剪贴板监控软件。

### Q: 中文显示乱码？
**A:** 确保目标程序支持 UTF-8 编码。

## 📊 技术细节

### HTML 结构
```
.modal#helpModal
  └── .modal-content
      ├── .modal-header (标题)
      ├── .help-content
      │   ├── .help-section (快速开始)
      │   ├── .help-section (操作类型)
      │   ├── .help-section (使用示例)
      │   ├── .help-section (注意事项)
      │   ├── .help-section (常见问题)
      │   └── .help-section (按键参考)
      └── .modal-actions (关闭按钮)
```

### CSS 类说明
- `.help-section`：章节容器
- `.help-title`：章节标题（橙色，下划线）
- `.operation-grid`：操作类型网格
- `.operation-item`：单个操作卡片
- `.example-box`：示例容器（左侧橙色边框）
- `.step-badge`：步骤徽章（橙色圆角）
- `.warning-list`：注意事项列表
- `.faq-item`：FAQ 项目
- `.keyboard-grid`：键盘网格
- `.keyboard-item`：单个按键

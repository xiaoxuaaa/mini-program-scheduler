# 定时点击助手

一个基于 Python 的命令行工具，帮助你在指定时间自动执行点击操作。

## 功能特性

- ✅ 定时任务调度（精确到分钟）
- ✅ 基于坐标的自动点击
- ✅ 任务配置持久化保存
- ✅ 多任务管理（添加、删除、启用/禁用）
- ✅ 鼠标坐标获取辅助工具
- ✅ 通用工具，适用于各种应用场景

## 安装

### 1. 确保已安装 Python 3.7+

```bash
python --version
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

依赖包说明：
- `pyautogui` - 自动化鼠标操作
- `schedule` - 定时任务调度
- `pywin32` - Windows 窗口管理
- `colorama` - 命令行彩色输出

## 使用方法

### 启动程序

```bash
python main.py
```

### 主要功能

#### 1. 添加任务

1. 在主菜单选择 `1. 添加任务`
2. 输入任务名称（如：抢购商品）
3. 输入任务描述（可选）
4. 输入执行时间（格式：`HH:MM`，如 `14:30`）
5. 输入点击坐标（使用坐标获取工具获得准确坐标）
6. 输入 `done` 完成配置

#### 2. 获取鼠标坐标

1. 在主菜单选择 `5. 获取鼠标坐标`
2. 按提示移动鼠标到目标位置
3. 记录显示的坐标值

**建议流程：**
- 先打开微信小程序到目标页面
- 使用坐标工具获取"加购"按钮等的坐标
- 记录所有需要点击的坐标
- 添加任务时输入这些坐标

#### 3. 启动调度器

1. 在主菜单选择 `6. 启动调度器`
2. 程序将在后台等待，到点自动执行任务
3. 按 `Ctrl+C` 停止调度器

## 配置文件

任务配置保存在 `config/tasks.json`，格式如下：

```json
[
  {
    "id": 1,
    "name": "商品抢购",
    "time": "14:30",
    "description": "XX 商品定时抢购",
    "clicks": [
      [500, 600],
      [500, 700]
    ],
    "enabled": true,
    "created_at": "2026-06-16T15:00:00",
    "last_run": null
  }
]
```

## 使用场景示例

### 场景 1：小程序商品定时抢购

1. 提前打开微信小程序到商品页面
2. 使用坐标工具获取"立即购买"按钮坐标

1. 打开浏览器到目标页面
2. 获取刷新按钮坐标
3. 设置定时刷新间隔
4. 保持页面活跃

### 场景 4：办公自动化

1. 打开需要定时操作的软件
2. 获取操作按钮坐标
3. 设置执行时间
4. 自动完成重复操作

## 注意事项

⚠️ **重要提醒**

1. **仅供学习和个人使用**，请勿用于商业目的或违规操作
2. 使用前请确保符合目标应用的使用条款
3. 建议合理控制频率，避免被检测为异常
4. 执行任务时需保持目标应用窗口在前台或可访问状态
5. 坐标是固定的，如果窗口大小或位置改变，需要重新获取坐标
6. 到达任务时间后会立即点击当前屏幕坐标，请确保目标应用已就绪

## 常见问题

### Q: 任务没有执行？
A: 检查任务是否启用，时间格式是否正确，调度器是否在运行。

### Q: 点击位置不准确？
A: 重新获取坐标，确保目标窗口位置和大小与设置时一致。

### Q: 可以同时执行多个任务吗？
A: 可以，只要时间不冲突即可。

### Q: 支持后台执行吗？
A: 到达任务时间时会直接点击屏幕坐标，建议将目标应用窗口保持在前台或确保坐标位置可点击。

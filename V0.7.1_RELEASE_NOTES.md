## ✨ 新增功能

### 📋 粘贴操作 (paste)

#### 功能特性
- ✅ 支持**中文**、日文、韩文等 Unicode 字符
- ✅ 自动复制到剪贴板并粘贴
- ✅ 解决文本输入只支持英文的问题
- ✅ 适用于密码、中文表单、特殊字符等场景

#### 工作原理
```python
1. 将文本复制到系统剪贴板
2. 模拟 Ctrl+V 快捷键粘贴
3. 完成输入
```

#### 使用场景
- **中文表单填写** - 姓名、地址等
- **密码输入** - 特殊字符密码
- **多语言内容** - 日文、韩文、emoji 等
- **复杂文本** - 包含特殊符号的内容

## 🔧 技术改进

### 后端增强 (`src/clicker.py`)

#### 新增方法
```python
def paste_text(self, text: str):
    """粘贴文本（支持中文等 Unicode 字符）"""
    pyperclip.copy(text)  # 复制到剪贴板
    pyautogui.hotkey('ctrl', 'v')  # 粘贴
```

#### 操作类型支持
```python
elif action_type == 'paste':
    text = action_data.get('text', '')
    self.paste_text(text)
```

### 前端增强 (`templates/index.html`)

#### 新增"粘贴"按钮
- 位置：操作编辑器工具栏
- 图标：📋
- 颜色：青色徽章

#### 粘贴表单
```html
<input type="text" placeholder="要粘贴的文本（支持中文）">
```

### JavaScript 更新 (`static/app.js`)

#### 添加粘贴操作支持
```javascript
case 'paste':
    return `📋 粘贴: "${action.text}"`;
```

## 📊 操作类型对比

| 操作类型 | 支持字符 | 速度 | 适用场景 |
|---------|---------|------|---------|
| **type** (输入) | 仅英文 | 慢（模拟打字） | 英文表单、搜索 |
| **paste** (粘贴) | 全部 Unicode | 快（瞬间粘贴） | 中文、密码、特殊字符 |

## 💡 使用示例

### 示例 1: 中文表单填写
```json
{
  "actions": [
    {"type": "click", "x": 500, "y": 300},
    {"type": "paste", "text": "张三"},
    {"type": "hotkey", "key": "tab"},
    {"type": "paste", "text": "北京市朝阳区"},
    {"type": "hotkey", "key": "enter"}
  ]
}
```

### 示例 2: 密码输入
```json
{
  "actions": [
    {"type": "click", "x": 600, "y": 400},
    {"type": "paste", "text": "P@ssw0rd!#2024"},
    {"type": "hotkey", "key": "enter"}
  ]
}
```

### 示例 3: 混合中英文
```json
{
  "actions": [
    {"type": "type", "text": "username"},
    {"type": "hotkey", "key": "tab"},
    {"type": "paste", "text": "这是中文内容！😊"}
  ]
}
```

# 快速开始指南

## 第一次使用教程

### 步骤 1: 安装依赖（仅首次需要）

```bash
pip install -r requirements.txt
```

### 步骤 2: 启动程序

```bash
python main.py
```

### 步骤 3: 获取点击坐标

假设你要在小程序中抢购商品，需要点击"立即购买"按钮：

1. 打开微信，进入目标小程序
2. 在程序主菜单输入 `5`（获取鼠标坐标）
3. 按回车，快速移动鼠标到"立即购买"按钮上
4. 等待 3 秒倒计时
5. 记录显示的坐标，例如：`720,850`

如果需要多次点击（例如先点商品详情，再点购买），重复上述步骤获取多个坐标。

### 步骤 4: 添加任务

1. 在主菜单输入 `1`（添加任务）
2. 按提示输入：
   ```
   任务名称: 抢购限量商品
   任务描述: 下午2点开售的限量商品
   执行时间: 14:00
   坐标 1: 720,850
   坐标 2: done
   ```

### 步骤 5: 查看任务

输入 `2` 查看所有已配置的任务，确认信息正确。

### 步骤 6: 启动调度器

1. 输入 `6` 启动调度器
2. 程序会显示下次执行时间
3. 保持程序运行，可以最小化命令行窗口
4. 确保微信保持打开状态（可以最小化）

### 步骤 7: 等待自动执行

- 到达设定时间后，程序会自动：
  1. 找到微信窗口并激活
  2. 按照配置的坐标依次点击
  3. 显示执行结果

## 实用技巧

### 技巧 1: 测试坐标准确性

添加任务后，可以手动触发一次测试：
- 设置一个 1-2 分钟后的时间
- 启动调度器观察效果
- 如果位置不对，重新获取坐标

### 技巧 2: 提前准备

- 在执行时间前 5-10 分钟启动程序
- 确保微信已打开并停留在小程序页面
- 可以最小化窗口，但不要关闭

### 技巧 3: 备用坐标

如果担心一次点击不成功，可以添加多个相同坐标：
```
坐标 1: 720,850
坐标 2: 720,850  # 重复点击
坐标 3: done
```

### 技巧 4: 多步操作

某些场景需要多步操作，例如：
1. 点击商品进入详情页
2. 点击"立即购买"
3. 点击"确认支付"

按顺序获取所有坐标，然后一次性配置到任务中。

## 故障排除

### 问题：程序找不到微信窗口
**解决方案：**
- 确保微信已打开
- 检查任务管理器中是否有微信进程
- 尝试重启微信

### 问题：点击位置偏移
**解决方案：**
- 微信窗口大小/位置改变了，重新获取坐标
- 使用全屏或固定窗口大小
- 确保 Windows 缩放比例未改变（建议 100%）

### 问题：任务没有执行
**解决方案：**
- 检查任务是否启用（输入 `2` 查看）
- 确认时间格式正确（必须是 HH:MM）
- 确认调度器正在运行
- 查看命令行是否有错误信息

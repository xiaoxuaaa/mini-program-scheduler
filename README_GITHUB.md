# 🤖 定时点击助手

一个功能强大的自动化工具，支持定时执行鼠标点击和键盘操作。

## ✨ 核心功能

### 5 种操作类型

- 🖱️ **点击** - 鼠标单击/双击
- ⌨️ **输入** - 英文文本输入
- 📋 **粘贴** - 中文文本粘贴（支持中文！）
- 🔘 **按键** - 键盘快捷键
- ⏱️ **等待** - 延时等待

### 任务管理

- ✅ 可视化 Web UI 界面
- ✅ 多时间点任务支持
- ✅ 上下移动操作顺序
- ✅ 实时任务调度
- ✅ 任务启用/禁用

## 🎯 适用场景

- 📱 小程序抢课/抢购
- 🎮 游戏自动签到
- 💼 办公自动化
- 🌐 网页定时操作
- 🔄 重复性任务自动化

## 📦 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动程序

```bash
python app.py
```

### 3. 打开浏览器

访问：http://localhost:5000

### 4. 创建任务

1. 点击"添加任务"
2. 选择"多时间点任务"
3. 添加操作（点击、输入、粘贴等）
4. 设置执行时间
5. 启动调度器

## 🔧 系统要求

- **Python**: 3.7+
- **操作系统**: Windows（主要支持）
- **浏览器**: Chrome/Edge/Firefox

## ⚠️ 注意事项

1. **文本输入限制**：`输入`操作仅支持英文，中文请使用`粘贴`操作
2. **窗口焦点**：执行时目标窗口需要在前台
3. **分辨率**：坐标与屏幕分辨率相关，更换电脑需重新获取坐标
4. **安全性**：密码等敏感信息会保存在配置文件中，注意保管

## 🔐 安全建议

如果部署到公网，建议：
- 添加用户登录验证
- 使用 HTTPS 加密
- 限制访问 IP
- 不要保存敏感密码

## 🐛 故障排除

### Q: 按钮点击无效？
**A**: 按 Ctrl+Shift+R 强制刷新浏览器缓存

### Q: 粘贴中文失败？
**A**: 确保使用`粘贴`操作而不是`输入`操作

### Q: 坐标不准确？
**A**: 使用"获取坐标"功能重新获取精确位置

# 任务编辑功能 - 使用指南

## 🎉 新增功能：编辑已有任务

现在可以直接编辑已创建的任务，无需删除重建！

---

## 功能说明

### 可以编辑的内容

✅ **基本信息**
- 任务名称
- 任务描述

✅ **时间配置**
- 简单任务的执行时间
- 多时间点任务的各个时间点

✅ **坐标配置**
- 添加新坐标
- 删除已有坐标
- 修改坐标值
- 调整坐标顺序（通过删除重新添加）
- 切换单击/双击

✅ **高级选项**
- 点击间隔
- 回到桌面选项

---

## 如何编辑任务

### 步骤 1：找到任务

在任务列表中找到要编辑的任务

### 步骤 2：点击"编辑"按钮

每个任务卡片上都有 **"编辑"** 按钮

### 步骤 3：修改内容

模态框会弹出，所有信息已自动填充：
- 修改任务名称
- 修改描述
- 修改时间
- 添加/删除/修改坐标
- 调整双击选项

### 步骤 4：保存修改

点击 **"更新任务"** 按钮保存

---

## 编辑示例

### 示例 1：修改坐标

**场景**：微信图标位置变了

**操作**：
1. 点击任务的"编辑"按钮
2. 找到微信图标的坐标
3. 修改 X、Y 值
4. 或者删除旧坐标，点击"获取当前位置"重新获取
5. 点击"更新任务"

### 示例 2：添加新的点击步骤

**场景**：需要多点一个按钮

**操作**：
1. 点击"编辑"
2. 点击"+ 添加坐标"
3. 输入新坐标或使用"获取当前位置"
4. 点击"更新任务"

### 示例 3：修改点击顺序

**场景**：需要调整点击顺序

**操作**：
1. 点击"编辑"
2. 删除需要移动的坐标（点击 × 按钮）
3. 在正确位置重新添加坐标
4. 点击"更新任务"

### 示例 4：改为双击

**场景**：发现需要双击才能触发

**操作**：
1. 点击"编辑"
2. 找到对应坐标
3. ☑ 勾选"双击"
4. 点击"更新任务"

### 示例 5：添加新时间点

**场景**：多时间点任务需要增加步骤

**操作**：
1. 点击"编辑"
2. 点击"+ 添加时间点"
3. 配置新时间点
4. 点击"更新任务"

## 界面说明

### 编辑模式 vs 创建模式

**创建模式**：
- 标题："添加新任务"
- 按钮："创建任务"
- 表单为空

**编辑模式**：
- 标题："编辑任务"
- 按钮："更新任务"
- 所有信息已填充

## 完整操作流程

### 简单任务编辑

1. **点击"编辑"按钮**
2. **修改内容**：
   - 任务名称
   - 执行时间
   - 坐标列表
3. **调整坐标**：
   - 修改现有坐标
   - 删除不需要的（点击 ×）
   - 添加新坐标（点击"+ 添加坐标"）
   - 获取新位置（点击"获取当前位置"）
4. **保存**：点击"更新任务"

### 多时间点任务编辑

1. **点击"编辑"按钮**
2. **修改基本信息**：
   - 任务名称
   - 任务描述
3. **编辑各个时间点**：
   - 修改步骤名称
   - 修改执行时间
   - 调整点击间隔
   - 切换"回到桌面"选项
   - 编辑坐标列表
4. **管理时间点**：
   - 删除时间点（点击时间点右上角 ×）
   - 添加时间点（点击"+ 添加时间点"）
5. **保存**：点击"更新任务"

## 技术实现

### 后端 API

**获取任务详情**：
```
GET /api/tasks/{task_id}
```

**更新任务**：
```
PUT /api/tasks/{task_id}
```

### 前端逻辑

1. 点击"编辑"按钮
2. 调用 API 获取任务详情
3. 填充表单（包括坐标、双击选项）
4. 用户修改
5. 点击"更新任务"
6. 调用 PUT API 更新

---

## 注意事项

### ✅ 可以做的

- 修改任何信息
- 添加/删除坐标
- 调整坐标顺序
- 切换单击/双击
- 添加/删除时间点

### ⚠️ 注意事项

- **调整顺序**：需要删除后重新添加到正确位置
- **保存后生效**：修改后需要点击"更新任务"才会保存
- **调度器影响**：如果调度器正在运行，建议停止后再编辑

## 常见场景

### 场景 1：坐标漂移

**问题**：屏幕分辨率改变，坐标不准了

**解决**：
1. 编辑任务
2. 逐个重新获取坐标
3. 更新任务

### 场景 2：流程调整

**问题**：操作流程变了，需要增加步骤

**解决**：
1. 编辑任务
2. 添加新的时间点或坐标
3. 更新任务

### 场景 3：测试优化

**问题**：点击间隔不合适

**解决**：
1. 编辑任务
2. 调整"点击间隔"数值
3. 更新任务
4. 测试效果

## 最佳实践

### 编辑前

1. ✅ 停止调度器
2. ✅ 确认需要修改的内容
3. ✅ 准备好新的坐标（如需要）

### 编辑中

1. ✅ 仔细核对每个坐标
2. ✅ 确认时间点顺序
3. ✅ 检查双击选项
4. ✅ 验证点击间隔

### 编辑后

1. ✅ 创建测试任务验证
2. ✅ 或修改为2分钟后的时间测试
3. ✅ 确认无误后改回正式时间
4. ✅ 启动调度器

## 快捷操作

### 快速修改坐标

1. 点击"编辑"
2. 删除旧坐标（点击 ×）
3. 点击"获取当前位置"
4. 等待 10 秒倒计时
5. 自动填入新坐标
6. 点击"更新任务"

### 批量调整

如果需要修改多个任务：
1. 逐个编辑
2. 或导出配置文件统一修改（高级）

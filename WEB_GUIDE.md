# 定时点击助手 Web 版使用指南

## 快速开始

### 1. 启动 Web 服务

```bash
python app.py
```

启动后会显示：

```
============================================================
         定时点击助手 Web 版 v0.5.0
============================================================

访问地址:
  本机: http://localhost:5000
  局域网: http://<你的IP>:5000

提示: 使用 ipconfig 查看本机 IP 地址
```

### 2. 访问界面

**本机访问**：打开浏览器访问 `http://localhost:5000`

**手机访问**（局域网）：
1. 确保手机和电脑连接同一个 WiFi
2. 在电脑上运行 `ipconfig`，找到 IPv4 地址（如 `192.168.1.100`）
3. 在手机浏览器输入 `http://192.168.1.100:5000`

## 功能说明

### 控制面板

- **启动调度器** - 开始自动执行任务
- **停止调度器** - 停止自动执行
- **添加任务** - 创建新的定时点击任务
- **状态指示器** - 实时显示调度器运行状态（绿点闪烁表示运行中）

### 任务管理

- **查看任务** - 显示所有任务及其状态
- **启用/禁用** - 快速切换任务状态
- **删除任务** - 移除不需要的任务
- **查看历史** - 显示任务上次执行时间

### 添加任务

1. 点击"添加任务"按钮
2. 填写任务信息：
   - **任务名称**（必填）
   - **执行时间**（必填，24小时格式）
   - **任务描述**（可选）
3. 添加点击坐标：
   - **手动输入**：直接输入 X 和 Y 坐标
   - **获取当前位置**：点击按钮获取当前鼠标位置
   - 可添加多个坐标，按顺序执行
4. 点击"创建任务"

## 注意事项

1. **调度器精度**：保持 0.3 秒轮询间隔（已配置）
2. **后台运行**：Web 服务启动后，调度器在后台线程运行
3. **配置共享**：Web 版和命令行版共享同一个 `config/tasks.json` 文件
4. **外网访问**：默认只支持局域网，如需外网访问需要内网穿透或云部署

## 常见问题

**Q: 手机无法访问？**
- 检查是否连接同一 WiFi
- 确认防火墙是否允许 5000 端口
- 尝试在电脑浏览器先访问 `http://localhost:5000` 确认服务正常

**Q: 坐标获取不准确？**
- 使用"获取当前位置"功能获取精确坐标
- 或在目标位置截图，使用图片工具查看坐标

**Q: 任务没有执行？**
- 确认调度器已启动（状态指示器显示绿点闪烁）
- 确认任务状态为"启用"
- 检查任务执行时间是否设置正确

## 技术栈

- **后端**：Flask + Python
- **前端**：原生 HTML/CSS/JavaScript（无框架依赖）
- **调度**：schedule 库（0.3秒轮询）
- **点击**：PyAutoGUI
